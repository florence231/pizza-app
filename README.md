# Pizza-app

# Requirement

> Ionic 5

# Comment migrer vers Ionic 5
``
 npm uninstall -g ionic

 npm install -g ionic # sudo peut être requis 
 ``

# Déploiement de l'application

> npm install // 1

> ionic serve

# En cas de problème pour 1

> npm cache clean -f

> npm install -g npm

> npm cache verify
